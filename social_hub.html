Application Under Development

This is an application to find the restaurants (searched by the name, type of food served or location), the bike docks near the selected restaurant and the availability at each dock.

The project will utilize UML in modeling the design blueprint of a real-time web app and use the following technologies in the implementation:

  1. JavaScript and TypeScript
  2. Node.js , Express, and Angular
  3. Material Design
  4. Angular Google Maps for Geospacial queries
  5. Python
  6. SQL (PostgreSQL) and NoSQL (ElasticSearch)
  7. Yelp and Divvy APIs for realtime data streaming
Sample developed by Prof. Atef Bader
