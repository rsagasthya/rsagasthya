McAfee, LLC is an American global computer security software company headquartered in Santa Clara, California and claims to be the world's largest dedicated security technology company.

McAfee primarily develops digital-security tools for personal computers and server devices, and more recently, for mobile devices.

McAfee brands, products and sub-products include:

  1. IntruShield
  2. McAfee Anti-Malware Engine Core (AMCore)
  3. McAfee Change Control
  4. McAfee DAT Reputation
  5. McAfee E-Business Server
  6. McAfee Entercept
  7. McAfee SiteAdvisor
  8. McAfee VirusScan
